# KryptoMin
Application for calculating cryptocurrencies taxes in Poland.

Technologies, techniques etc. used in project:
- .NET 6.0
- Azure Functions
- Azure Table Storage
- Azure Blob Storage
- Azure Key Vault
- Azure Static Web App
- xUnit
- Fluent Assertions
- Angular 14
- Angular Material

See the app: https://www.krypto-min.pl/
